Welcome to the World of HTML & CSS!

Introduction

What is HTML?

HTML is short for HyperText Markup Language. It is a way to explain to a computer the semantic layout of a webpage.

What is CSS?

CSS is short for Cascading StyleSheet. CSS is used to define the visual look of an HTML page.

HTML Basics

What is a tag?

An HTML tag is either the opening or closing part of an element. Examples of tags are "h2" "h1" or "p". Open tags can contain attributes specifiying infoormation about the element. Closing tags contain a '/'. Some elements are self-contained.

What is an element?

An HTML element is defined by an opening and closing tag (unless the element is self-closing). Elements are the components that make up an HTML page, being paragraphs, headings, the body, or links. Some elements are in-line, meaning they won't make a new line, others are block elements meaning they will create a new line.

What is an attribute?

An HTML attribute is placed within an element to add more information to it. Examples include: "for", "style", "id", and "class".

CSS Basics

What is a selector?

A CSS indicator of what you are styling. Selectors can focus on: ids, classes, and element.

What is a class ?

A class can be applied to an HTML element to specify styling. The advantage of using classes is the ability to apply styling to multiple different types of elements.

What is an ID?

An ID identifies a unique element. This can be used to find an element later or specify a unique styling.
